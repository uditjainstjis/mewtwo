Research Discovery

The Code Paradox: An Unexpected Breakthrough

During evaluation, we discovered something no one predicted: the code-trained adapter outperforms the math-trained adapter on mathematical reasoning by +10.9%.

MATH-500 Benchmark
Baseline (no adapter)41.5%
Math adapter50.5%
Code adapter 🔥56.0%

Why This Matters

Code training teaches structured reasoning — variable binding, logical sequencing, decomposition — which transfers powerfully to mathematical problem-solving.

This suggests that domain expertise is more transferable than previously believed, and that Synapta's modular routing can discover these cross-domain synergies automatically.

This is a publishable finding with implications for how we train specialized AI systems.
